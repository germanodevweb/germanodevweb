# 💼 Projeto em Destaque

Meu principal projeto é o **Portal Imobiliário 3 Pinheiros**, uma aplicação **Full Stack** desenvolvida para solucionar uma necessidade real da empresa da minha família.

Ao invés de utilizar uma plataforma pronta, decidi projetar e desenvolver uma solução própria, desde a definição da arquitetura até a implementação completa do sistema.

Antes de escrever qualquer linha de código, desenhei toda a interface da aplicação, planejando a experiência do usuário, a organização das informações e a navegação, buscando criar um sistema intuitivo, rápido e de fácil utilização.

A arquitetura foi planejada utilizando **Next.js**, **React**, **TypeScript**, **PostgreSQL** e **Prisma ORM**, priorizando desempenho, escalabilidade, organização do código e boas práticas de SEO para melhorar a indexação nos mecanismos de busca.

O projeto reúne um **portal imobiliário público**, um **CRM administrativo**, uma **base integrada de leads**, um **blog otimizado para SEO** e recursos de **Inteligência Artificial** para auxiliar na criação de descrições de imóveis e artigos.

Durante todo o desenvolvimento utilizei o **Cursor AI** como ferramenta de apoio para acelerar a implementação, permanecendo sob minha responsabilidade todas as decisões relacionadas à arquitetura, experiência do usuário, organização do código, implementação, testes, correções e evolução contínua do sistema.

Hoje o projeto encontra-se em produção, sendo utilizado como plataforma oficial da empresa e recebendo melhorias constantes.

# 🚀 Principais Funcionalidades

O Portal Imobiliário 3 Pinheiros foi desenvolvido como uma solução completa para o mercado imobiliário,reunindo um portal público, um CRM administrativo e recursos de Inteligência Artificial para otimizar a operação imobiliária.

Entre as principais funcionalidades estão:

- Cadastro completo de imóveis.
- Painel administrativo para gerenciamento dos imóveis.
- CRM integrado para gestão de clientes e leads.
- Captação automática de leads através dos formulários do portal.
- Base integrada para acompanhamento de clientes e oportunidades de negócio.
- Gestão completa do Blog Imobiliário.
- Geração assistida por IA para descrições de imóveis.
- Apoio de IA para criação de artigos otimizados para SEO.
- Blog integrado para estratégia de marketing de conteúdo.
- Sitemap XML gerado automaticamente.
- Integração com Google Analytics 4.
- Integração com Google Search Console.
- Estrutura preparada para Schema.org.
- Upload e gerenciamento de imagens.
- Layout responsivo para desktop e dispositivos móveis.
- Aplicação em produção e utilizada em ambiente real.

# 🏗️ Arquitetura da Solução

Antes do desenvolvimento defini uma arquitetura que permitisse criar uma aplicação organizada, escalável e preparada para evolução contínua.

Cada tecnologia foi escolhida de acordo com sua responsabilidade dentro do projeto.

### Next.js

Responsável pela estrutura da aplicação, renderização otimizada, Server Components e recursos voltados para SEO.

### React

Utilizado para construção da interface através de componentes reutilizáveis, proporcionando organização e facilidade de manutenção.

### TypeScript

Adicionado para aumentar a segurança do código, reduzir erros durante o desenvolvimento e facilitar futuras evoluções.

### PostgreSQL

Banco de dados relacional utilizado para armazenar imóveis, artigos, usuários, imagens e leads.

### Prisma ORM

Responsável pela comunicação entre a aplicação e o banco de dados, tornando as consultas mais seguras e produtivas.

### Tailwind CSS

Utilizado para construção rápida de interfaces modernas, responsivas e padronizadas.

### Cloudinary

Responsável pelo armazenamento e otimização das imagens dos imóveis.

### Vercel

Plataforma utilizada para hospedagem, deploy contínuo e distribuição da aplicação em produção.
